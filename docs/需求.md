
本项目 需要写一个 标准的 网页图片 抓取   sdk  也就是 packagist 包

用php 7.4 写

包名为 zyan/u2p

需要专项 攻克 指定的 网站 中的图片  

输入 网页链接  返回 图片 链接 数组

需要你自己摸索 拿 到 网页中的图片  其中网页中的图片 可能是通过 ajax 接口 获取的 也可能是 直接 通过 网页源码 获取的
